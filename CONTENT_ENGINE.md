## 🔄 8-PHASE WORKFLOW

### PHASE 1: Analysis (4 Components)
1. **Local Competitor Weaknesses** — Scan Belleva, Shine, VIP, etc. for gaps
2. **Customer Review Complaints** — Extract recurring pain points
3. **Global Nail Trends** — TikTok, Instagram, NYFW, Seoul trends
4. **Existing Blog Audit** — Check BlogPage.tsx for covered topics

### PHASE 2: Opportunity Identification
- Find ONE strategic blog opportunity not yet covered
- Must address competitor weakness + trend alignment + local SEO value

### PHASE 3: Blog Proposal
- Create proposal document
- **WAIT FOR YOUR APPROVAL** before proceeding

### PHASE 4: Approval Gate
- You review proposal
- Approve, modify, or reject
- Only proceed to writing after approval

### PHASE 5: Full Article Creation
- 800-1,200 words
- SEO optimized
- Meta title + description
- Internal links
- Image placement suggestions

### PHASE 6: Publishing Gate
- **DO NOT AUTO-PUBLISH**
- Wait for your final approval

### PHASE 7: Social Content Blueprint
- Instagram post concept
- Reel/TikTok video concept
- 3 hook options
- Caption draft
- Hashtags

### PHASE 8: Ad Promotion Plan
- Best posting times
- Boosting strategy
- Audience targeting
- Ad copy variations

### How to Approve:

- **"Approved — proceed with writing"** → Creates full article
- **"Modify: [your changes]"** → Updates proposal, waits again
- **"Skip this week"** → Waits until next Tuesday

### How to Publish:
After article is written:
- Review the draft
- Reply **"Publish to blog"** → I'll create GitHub PR
